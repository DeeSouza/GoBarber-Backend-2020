> Aplicação para agendamento de serviços de barbearia.

## Install

    yarn

## Run

    yarn dev:server

## What I Used

- NodeJS (Express)
- TypeORM
- Redis
- Multer (Upload de Arquivos)
- Json Web Tokens (Autenticação JWT)

## My Setup DEV

- Editor Config
- ESLint
- Prettier

## Docker

    docker run --name gobarber_postgres -e MYSQL_ROOT_PASSWORD=docker -d postgres
